## Setting up a Development Environment

### Linux / MacOS
```sh
cd PROJECT_DIR
# Install pre-commit
pre-commit install
```

### NixOS

On NixOS, the dev environment will set itself up upon `cd`-ing into the directory.

```sh
cd PROJECT_DIR
# If Direnv is allowed
direnv allow
# Or
nix-shell
```
